## About

I build and maintain Docker-based, zero-installation Blue Team lab environments at **[AltaySec Labs](https://altaysec.com.tr)** — open-source SOC training scenarios designed for hands-on detection engineering and threat hunting practice.

My focus: simulating real-world attack techniques and engineering the detection logic to catch them.

- Threat Detection & SIEM Engineering (Wazuh + Sysmon)
- Detection-as-Code — Sigma rules mapped to MITRE ATT&CK
- Purple Team operations — attack simulation + defensive coverage
- Cloud security monitoring (AWS CloudTrail / IAM breach detection)
- Incident response automation & SOAR workflows
